## Create symklinks

Create base directory for symlinks and move into it.

```shell
mkdir -p ~/bin/git
cd ~/bin/git
```

```shell
#!/bin/bash
#
# file: activate-git-2.9.2.sh
#

rm bin
ln -s ../git-2.9.2/bin bin

rm lib
ln -sf ../git-2.9.2/lib lib

rm libexec
ln -sf ../git-2.9.2/libexec libexec


rm share
ln -sf ../git-2.9.2/share share
```
